# 🖼️ Images 사용법

## 귀손눈.png 사용하기

### 위치
`x/Images/귀손눈.png`

### 사용법

**1. 기본 삽입**
```markdown
![[귀손눈.png]]
```

**2. 크기 조절**
```markdown
![[귀손눈.png|200]]     # 200px 너비
![[귀손눈.png|300]]     # 300px 너비
```

**3. 테이블에서**
```markdown
| 👂 귀 | ✋ 손 | 👁 눈 |
|:---:|:---:|:---:|
| Gap | Important | Valid |

![[귀손눈.png|150]]
```

---

## 현재 사용 위치

1. **Home.md** - 메인 화면 상단
2. **논문 3질문 프레임워크.md** - 설명용
3. **Paper Template** - (선택사항)
