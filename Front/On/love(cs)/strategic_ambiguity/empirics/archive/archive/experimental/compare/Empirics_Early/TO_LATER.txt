## ðŸ” Variable Definitions (for validation)

### **Vagueness Index Construction**

**Source:** Composite of two text-based measures from PitchBook `description` field

**Component 1: Categorical Vagueness**
```python
abstract_keywords = [
    'platform', 'solution', 'ecosystem', 'framework', 
    'innovative', 'next-generation', 'cutting-edge',
    'transformative', 'revolutionize', 'disruptive'
]
categorical_vagueness = sum(1 for word in description.lower().split() 
                             if word in abstract_keywords)
```

**Component 2: Concreteness Deficit**
```python
specific_patterns = [
    r'\d+',                    # Numbers (e.g., "100 qubits")
    r'\d{4}',                  # Years (e.g., "by 2025")
    r'(qubit|nm|GHz|TB|ms)',  # Technical units
]
specific_count = sum(len(re.findall(pattern, description)) 
                      for pattern in specific_patterns)
concreteness_deficit = 1 - (specific_count / len(description.split()))
```

**Final Index:**
```python
vagueness_raw = (categorical_vagueness + concreteness_deficit) / 2
vagueness_zscore = (vagueness_raw - mean(vagueness_raw)) / std(vagueness_raw)
```

**Interpretation:**
- z = âˆ’2: Very precise (e.g., "5-qubit ion trap for drug discovery, FDA approval by 2024")
- z = 0: Average vagueness
- z = +2: Very vague (e.g., "Revolutionary quantum platform for next-gen solutions")
