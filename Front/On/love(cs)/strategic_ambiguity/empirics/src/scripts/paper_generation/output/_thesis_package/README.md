## One-Page Summary

### Research Question
When do the commitments that secure early resources become constraints on long-term success?

### Core Finding
Promise precision and venture growth exhibit a **U-shaped relationship**. Both highly precise and highly vague positioning outperform intermediate positioning.

### Mechanism: Golden Cage

1. **Capital requires commitment** ‚Üí specific promises
2. **Commitment attracts alignment** ‚Üí homogeneous stakeholders
3. **Alignment compresses variance** ‚Üí œÉ¬≤ ‚Üí 0
4. **Compressed variance blocks adaptation** ‚Üí rigidity

### Prescription

**For founders**: Select channel based on CR (cost ratio):
- **Analyst channel** (CR < 0.5): Precise promises, k* ‚âà 1
- **Believer channel** (CR > 0.5): Abstract vision, k* > 1
- **Never intermediate** ‚Üí survival penalty of 2-4 percentage points
