# 🤹N: 약속 상인

> **한 줄 요약**: "중간은 타협이 아니라 죽음이다."

---

## 🎯 핵심 메시지

| 통념 | 발견 |
|------|------|
| 중간이 안전하다 | 중간에서는 거래 자체가 성립 안 됨 |
| 균형을 찾으면 된다 | **최적해($k^*$)가 존재하지 않는 구간**이 있다 |

---

## 📊 Surprise: 해가 없는 구간

```
투자자
유형
  ↑
  │  Analyst ●────────────●  "구체적일수록 좋아"
  │           \          /
  │            \   ❌   /     ← 이 구간: 아무도 투자 안 함
  │             \ 진공 /
  │  Believer ●────────────●  "모호할수록 좋아"
  └─────────────────────────→ 모호함 (V)
        0     0.5      1
```

**V ≈ 0.5 (중간)**에서:
- Analyst: "검증 불가능한 도박"
- Believer: "꿈이 없는 지루한 사업"
- → **양쪽 다 투자 안 함** → 균형점 자체가 증발

---

## 🔬 수학적 증명

Analyst와 Believer의 이익 함수가 **반대 방향**:

| | Analyst | Believer |
|---|---------|----------|
| V 증가 시 | 이익 ↓ | 이익 ↑ |
| 미분 부호 | − | + |

**V ≈ 0.5**에서 두 미분값이 충돌 → **최적 $k^*$ 가 정의 불가**

> "투자가 실패하는 곳이 아니라, 투자라는 행위 자체가 성립할 수 없는 곳"

## 🎻 광기의 비유

| 논문 | 광기 유형 | 연주 비유 |
|------|-----------|-----------|
| **N** | 🌊 물광기 (ρ) | 숨 멈춤으로 **홀림** |

> 심사위원 반응: "논리적으로 반박이 불가능하다."

**왜 물광기인가?**
- 화려하지 않다 (빠르지도, 크지도 않음)
- 하지만 빈틈이 없다 (수학적 증명)
- 빠져나갈 구멍이 없어서 숨이 막힘
