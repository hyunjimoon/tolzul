## 📋 사용법

이 폴더에는 **지금 이 순간 집중하고 있는 작업**만 둡니다.

### 규칙
- **최대 3개**까지만 (이상적으로는 1-2개)
- 매일 또는 매주 작업하는 것들
- 명확한 다음 행동(next action)이 있어야 함
- 데드라인이 가까운 것들
