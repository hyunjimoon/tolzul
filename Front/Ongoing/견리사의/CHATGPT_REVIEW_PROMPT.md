## 🚀 당신의 임무 (10분 내)

### 1. 즉각적 판단

**질문**: 이 작업이 실행 가능한가?
- [ ] Yes - 즉시 시작 가능
- [ ] Yes, but - 조건부 가능 (우려사항 명시)
- [ ] No - 재검토 필요

### 2. Spec A 작성

```markdown
## 🚀 Spec A: 대담한 실행 계획

### 핵심 아이디어 (1-2문장)
[권준의 작업이 무엇을 달성하는가?]

### 즉시 실행 가능 항목
1. [백업 및 폴더 생성 - 난이도 ★☆☆]
2. [System Prompt 작성 - 난이도 ★★☆]
3. [협업 프로토콜 정의 - 난이도 ★☆☆]

### 예상 소요 시간
- Phase 0 (백업): X분
- Phase 1 (폴더): Y분
- Phase 2 (Prompt 작성): Z분
- **총**: [X+Y+Z]분

### 주요 리스크 (상위 3개만)
1. [리스크1]
2. [리스크2]
3. [리스크3]

### 권준에게 바로 전달할 것
- "일단 [X]부터 시작하라"
- "Phase 2에서 [Y]에 집중하라"
- "완벽보다 완성을 우선하라"

### 김완이 검증해야 할 것
- [ ] [항목1]
- [ ] [항목2]
- [ ] [항목3]
```

### 3. 속도 최적화 제안

**질문**: 어떻게 하면 더 빠르게 완성할 수 있는가?
```markdown
## ⚡ 속도 최적화

### 병렬 실행 가능한 것
- [작업A]와 [작업B]를 동시에

### 나중으로 미룰 수 있는 것
- [중요하지만 urgent하지 않은 것]

### 완벽주의 함정
- [과도하게 정교하게 만들 위험이 있는 부분]
```
