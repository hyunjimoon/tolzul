# "광기의 감각" 정의2

집요함을 자신감에서 추론할 수 있을 때

제가 영지님의 광기를 느낀 과정을 이론화하면
1. 영지님의 집요한 개선 노력이 자신감으로 축적 (x ->y)
2. 자신있는 연주가 특정 표현으로 제게 전달 (y -> y')
3. 전달된 표현에서 제가 집요함을 추론 (y'-> x')

도식: 영지's 집요 (x) -> 자신있는 표현 (y, y') <- 현지 (x')

실증 (why->how->what' vs what)
위 렌즈로 제 상황을 관찰하며 가지게 된 의심은, 광기에도 여러 종류가 있지 않을까? 영지님의 그리그 [1, 1']와 슈베르트 [2]에서 광기의 이유를 찾자면 빠른 표현 때문이에요.
