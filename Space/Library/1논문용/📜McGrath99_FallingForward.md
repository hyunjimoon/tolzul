# Falling Forward: Real Options Reasoning and Entrepreneurial Failure

## Summary
McGrath (1999) reconceptualizes entrepreneurial failure through real options lens: small failures are **option exercises** that generate learning, not managerial incompetence. "Falling forward" = controlled losses that redirect resources toward better opportunities.

## Research Question
How should we reframe entrepreneurial failure from a strategic perspective?

## Key Concepts

### Failure as Option Exercise
- Small investment = Option premium paid
- Failure outcome = Option expires worthless (but information gained)
- Learning captured = Residual value from "failed" option

### Falling Forward
The ability to convert failure into forward momentum:
1. Controlled loss magnitude
2. Information extraction from failure
3. Rapid redeployment to next opportunity

## Main Results

### Strategic Failure Design
Optimal entrepreneurial strategy involves **designing for failure**:
- Multiple small bets (buy many cheap options)
- Clear failure criteria (know when to abandon)
- Learning capture systems (extract information value)

### Value Preservation
Even "failed" ventures preserve value through:
- Market knowledge gained
- Team capabilities developed
- Network relationships built

## Critical Insights

> **Scott Stern Application**: This is the theoretical foundation for evidence-based entrepreneurshipâ€”experiments are valuable precisely because they can "fail forward."

> **Charlie Fine Application**: Operations design should build in "graceful degradation" paths where failure preserves redeployable resources.
