# Non-Additivity in Portfolios of Exploration Activities

## Summary
Vassolo, Anand & Folta (2004) prove that **multiple exploration options don't simply add up**. When options target overlapping outcomes, portfolio value is sub-additive (V(A+B) < V(A)+V(B)). When options share resources or complement each other, value becomes super-additive. Tested empirically on biotech alliances.

## Research Question
How do interactions among exploration options (e.g., R&D alliances) affect total portfolio value?

## Key Concepts

### Sub-Additivity
Options with **overlapping outcomes** (substitute technologies targeting same disease) cannibalize each other's valueâ€”if one succeeds, the other becomes redundant.

### Super-Additivity
Options with **shared resources** or **complementary outcomes** create synergiesâ€”shared learning infrastructure amplifies each option's value.

## Main Results

### Empirical Evidence (Biotech Alliances)
- Same therapeutic area â†’ Sub-additive (redundancy)
- Shared research platform â†’ Super-additive (synergy)
- Net effect depends on portfolio composition

### Optimal Portfolio Design
Not "more is better" but "fit matters"â€”design options to complement, not substitute.

## Critical Insights

> **Scott Stern Application**: When choosing among "equally viable alternatives," consider whether they cannibalize each other's learning value.

> **Charlie Fine Application**: Operational resource sharing can flip a sub-additive portfolio into super-additiveâ€”design infrastructure for option synergies.

## Methodology
**Approach**: Mathematical model + Empirical validation
**Data**: 200+ biotech-pharma alliances (1988-1999)
**Analysis**: Event study + regression on alliance portfolio value
