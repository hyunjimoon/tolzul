# 정체성이 만드는 관성의 덫
*Zuzul & Tripsas의 Flexibility: 창업자 정체성과 스타트업의 적응역량*

## 핵심 정의: Flexibility란 무엇인가?

### 학술적 정의 (Academic Definition)
Zuzul & Tripsas (2020)에서 **스타트업의 유연성(Flexibility)**은 단순히 '변하기 쉬움'이 아니라:

> **"불확실한 초기 시장에서 기회를 발견(Discover)하기 위해 끊임없이 실험하고 적응하는 역량"**

구체적으로:
- **전략적 재구성 능력**: "방향을 빠르게 전환하고(change direction quickly), 자원·고객·기술·비즈니스 모델을 전략적으로 재구성(reconfigure strategically)하는 능력" (Nadkarni & Herrmann, 2010: 1050)
- **지속적 실험**: "ongoing experimentation and adaptation"을 통해 구현
- **적응적 발견**: 환경 변화에 맞춰 최선의 경로를 찾아가는 과정

### 논문 맥락에서의 특징 (Contextual Nuance)

| 차원 | Flexibility (유연성) | Inertia (관성) |
|:---|:---|:---|
| **창업자 정체성** | Discoverer (발견자) | Revolutionary (혁명가) |
| **자기 인식** | "기회를 찾아내는 사람" | "세상을 바꾸는 사람" |
| **비즈니스 모델** | 지속적 실험과 적응 | 초기 모델에 고착 |
| **기술 선택** | 다양한 기술 유연하게 활용 | 특정 혁신 기술에 집착 |
| **생태계 투자** | 낮은 투자, 기존 인프라 활용 | 높은 투자, 직접 구축 |
| **범주 레이블** | 다양하고 유연한 명칭 사용 | 일관된 새 범주 주장 |

### 나의 논문과의 연관성 (Connection to My Thesis)

**"Flexibility and Commitment in Entrepreneurship"** 논문의 핵심 주장과 직접 연결:

1. **전략적 모호성의 두 얼굴**
   - Discoverer: 모호성을 유연성의 도구로 활용 → 다양한 이해관계자 조정 가능
   - Revolutionary: 명확한 비전에 집착 → 관성의 덫에 빠짐

2. **Scott Stern의 관점에서**
   - Discoverer는 "증거 기반 학습"을 통해 "전략적 선택"으로 수렴
   - 가설 검증 → 피봇 → 재검증의 반복

3. **Charlie Fine의 관점에서**
   - Discoverer의 유연성 = 운영 시스템의 적응성 확보
   - 초기 약속(commitment)을 낮게 유지 → 스케일링 리스크 관리

## 핵심 논점 (Core Argument)

스타트업이 일반적으로 유연하다고 알려진 것과 달리, 일부 스타트업은 강한 관성(Inertia)을 보인다. 그 근본 원인은 **창업자의 정체성(Founder Identity)**이다. 자신을 '혁명가(Revolutionary)'로 보는 창업자는 급진적 비전에 집착하여 관성에 빠지고, '발견자(Discoverer)'로 보는 창업자는 지속적 실험을 통해 유연성을 발휘한다.

## 🐅 핵심 이론 구조: 정체성 확인(Identity Affirmation) 메커니즘

### 두 가지 창업자 정체성 원형 (Two Founder Identity Archetypes)

```
┌─────────────────────────────────────────────────────────────────┐
│                    REVOLUTIONARY IDENTITY                        │
│  "나는 세상을 바꾸는 혁명가다"                                      │
├─────────────────────────────────────────────────────────────────┤
│  • 동기: 새로운 것(novel)을 창조                                   │
│  • 목표: 근본적 변화와 산업 형성(significant impact)                │
│  • 행동: 급진적 비전에 집착, 변화 거부                              │
│  • 결과: INERTIA (관성)                                           │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                     DISCOVERER IDENTITY                          │
│  "나는 기회를 발견하는 탐험가다"                                    │
├─────────────────────────────────────────────────────────────────┤
│  • 동기: 외부에서 창출된 기회 포착                                  │
│  • 목표: 성공적인 비즈니스 구축                                     │
│  • 행동: 지속적 실험과 적응                                        │
│  • 결과: FLEXIBILITY (유연성)                                      │
└─────────────────────────────────────────────────────────────────┘
```

### 정체성 확인(Identity Affirmation)의 4가지 채널

| 채널 | Revolutionary의 선택 | Discoverer의 선택 |
|:---|:---|:---|
| **1. 비즈니스 모델** | 급진적으로 새로운 모델 고수 | 기존 차터 모델 + 점진적 실험 |
| **2. 기술 활용** | 혁신적 기술(Eclipse 500)에 집착 | 다양한 기술 유연하게 채택 |
| **3. 생태계 투자** | 공항/인프라에 대규모 투자 | 기존 인프라 활용, 투자 최소화 |
| **4. 범주 상징** | 새로운 범주 레이블 일관되게 사용 | 다양한 레이블 상황에 맞게 사용 |

### 자기강화 순환 모델 (Self-Reinforcing Cycle)

```
FOUNDER IDENTITY
      │
      ▼
┌──────────────────────────────────────────────────────────────┐
│                    IDENTITY AFFIRMATION                       │
│  "정체성에 부합하는 행동을 함으로써 자아를 확인하고 유지"          │
└──────────────────────────────────────────────────────────────┘
      │                                    │
      ▼                                    ▼
┌────────────────────┐          ┌────────────────────┐
│   REVOLUTIONARY    │          │    DISCOVERER      │
│                    │          │                    │
│ Strong Commitments │          │ Low Commitments    │
│ to radical venture │          │ ongoing experiment │
│       concept      │          │   and adaptation   │
└────────────────────┘          └────────────────────┘
      │                                    │
      ▼                                    ▼
┌────────────────────┐          ┌────────────────────┐
│ AUDIENCE           │          │ AUDIENCE           │
│ EXPECTATIONS       │          │ EXPECTATIONS       │
│ Clear novelty &    │          │ Ambiguous          │
│ impact expected    │          │ expectations       │
└────────────────────┘          └────────────────────┘
      │                                    │
      ▼                                    ▼
   INERTIA                           FLEXIBILITY
 (Active reinforcement)           (Ongoing change)
      │                                    │
      └────────────► 정체성 강화 ◄──────────┘
```

## 사례 기업 비교: Air Taxi 산업 4개 기업

### 기업 개요

| 기업 | 창업자 정체성 | 설립연도 | 항공기 | 운명 |
|:---|:---|:---:|:---|:---|
| **DayJet** | Revolutionary | 2002 | Eclipse 500 | 2008 파산 |
| **SATSAir** | Revolutionary | 2004 | Cirrus SR-22 | 2009 파산 |
| **Linear Air** | Discoverer | 2004 | Cessna → Eclipse | 연구기간 내 생존 |
| **ImagineAir** | Discoverer | 2004 | Cirrus SR-22 | 연구기간 내 생존 |

### 비즈니스 모델 변화 비교

**Revolutionary 기업 (DayJet, SATSAir):**
- 초기 모델의 **강화(Reinforcement)**: 기존 모델에 투자 확대
- 변화 횟수: DayJet 1회 (파산 직전 임시), SATSAir 3회
- 특징: 차터 모델과의 차별화 고수

**Discoverer 기업 (Linear Air, ImagineAir):**
- 지속적 **실험과 적응**
- 변화 횟수: Linear Air 11회+, ImagineAir 9회+
- 특징: 차터 브로커 활용, 다양한 가격 전략 테스트

### 대표적 인용구

**DayJet 창업자 (Revolutionary):**
> "Nobody has ever done this before. That's how we got started."
> "We don't see this as incremental, another form of charter."

**Linear Air 창업자 (Discoverer):**
> "The basic approach is test and learn. What are you testing? What are you learning?"
> "We have been all over the road! But hey, it's all about being nimble in a market like ours."

## For Your Paper: 적용 가이드

### 1. "전략적 모호성" 연구에의 시사점

**이 논문이 말해주는 것:**
- 창업자 정체성이 **전략적 선택의 유연성**을 결정
- Revolutionary는 명확한 비전 → 모호성 기피 → 관성
- Discoverer는 탐색적 태도 → 모호성 활용 → 유연성

**당신의 논문에 적용:**
```
가설 제안:
H_identity: 창업자가 Discoverer 정체성을 가질수록, 
           가치 제안에서 전략적 모호성을 더 많이 활용한다.
```

### 2. 이해관계자 조정(Stakeholder Coordination) 관점

**Revolutionary의 딜레마:**
- 명확한 비전 → 특정 이해관계자에게 강력한 어필
- BUT 환경 변화 시 조정 불가 → 관계 경직

**Discoverer의 전략:**
- 모호한 가치 제안 → 다양한 해석 허용
- 환경 변화에 따라 재프레이밍 가능 → 유연한 조정

### 3. 실증 연구 설계 힌트

**종속변수 측정 아이디어:**
- 비즈니스 모델 변화 빈도
- 피봇(Pivot) 횟수
- 범주 레이블 변화 빈도

**조절변수 고려:**
- 산업의 불확실성 수준
- 초기 자원 보유량
- 창업자 이전 경험
