## Practical Checklists

### âœ… Sentence-Level Checklist

- [ ] Is the main verb an action verb (not is/are/was/were)?
- [ ] Is the subject concrete (not "it" or "there")?
- [ ] Does the sentence start with known information?
- [ ] Is it under 25 words? (If not, split)
- [ ] Remove "the fact that", "in order to", "it is important to note"

### âœ… Paragraph-Level Checklist

- [ ] Does the first sentence state the paragraph's main point?
- [ ] Does each sentence follow logically from the previous?
- [ ] Would removing transition words break the logic? (If yes, fix logic first)
- [ ] Is there exactly one idea per paragraph?
- [ ] Is the paragraph under 150 words?

### âœ… Section-Level Checklist

- [ ] Does the section title preview the content?
- [ ] Do subsection titles form a logical sequence?
- [ ] Is there a clear "They Say â†’ I Say" structure?
- [ ] Are contributions stated explicitly (not implied)?
- [ ] Is "So What" answered?

## Transition Logic (Not Just Words)

### âŒ Bad: Verbal Transition Only
```
Signaling theory suggests precision helps.
However, we find a U-shape.
Therefore, the theory needs modification.
```
â†’ "However" is doing all the work. WHY is there contrast?

### âœ… Good: Logical Transition
```
Signaling theory suggests precision helpsâ€”when investors verify claims.
But not all investors verify. Some project their vision onto abstract promises.
For projection-oriented investors, precision constrains rather than enables.
This heterogeneity explains the U-shape.
```
â†’ Each sentence sets up the next. Logic creates the transition.

### Test: Remove Transition Words
If removing "However/Therefore/Moreover" breaks comprehension, the logic is weak.
Fix the logic first, then add transition words only if needed.

## Management Science Style Guide

### Tone
| Avoid | Use |
|-------|-----|
| "We prove" | "Our evidence suggests" |
| "dramatically" | "$2.7\times$" |
| "toxic", "fatal" | "negatively associated" |
| "the murky middle" (repeatedly) | "intermediate positioning" |
| "I" | "We" (even for sole author) |

### Metaphors
- **One primary concept** is acceptable (e.g., "Promise Vendor")
- Multiple competing metaphors feel scattered
- Introduce formally: "a phenomenon we term the 'X'"

### Claims
- Association â‰  Causation (always acknowledge)
- Boundary conditions > Universal claims
- Effect sizes > Adjectives
