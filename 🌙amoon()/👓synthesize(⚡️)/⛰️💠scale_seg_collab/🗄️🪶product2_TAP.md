| Aspect | Horizontal Correlation ($k_h$) | Vertical Correlation ($k_v$) |
|--------|--------------------------------|----------------------------|
| **Nature of Constraint** | Spatial: Across contemporaneous players/technologies<br>$\color{Orange}{\rho_h}$ reveals current industry interdependencies | Temporal: Through value chain/developmental stages<br>$\color{Orange}{\rho_v}$ reveals historical path dependencies |
| **Environmental Setting** | $\color{Orange}{k_h}$ determines:<br>‚Ä¢ Industry-wide exploration space<br>‚Ä¢ Cross-fertilization opportunities<br>‚Ä¢ Complementary technology needs | $\color{Orange}{k_v}$ determines:<br>‚Ä¢ Sequential development constraints<br>‚Ä¢ Value chain rigidity<br>‚Ä¢ Historical path dependency |
| **Mathematical Impact** | Affects prediction variance:<br>$\color{Green}{\sigma_c} \sim \color{Orange}{\rho_h} * \color{SkyBlue}{exp(\sigma)}$ | Affects base probability:<br>$\color{Green}{p_c} \sim Beta(\alpha_c, \beta_c * \color{Orange}{\rho_v})$ |
| **Search Strategy When High** | ‚Ä¢ Need broad industry sampling<br>‚Ä¢ Focus on complementary technologies<br>‚Ä¢ Capture cross-player dynamics | ‚Ä¢ Need clean temporal samples<br>‚Ä¢ Focus on breaking path dependencies<br>‚Ä¢ Reveal value chain constraints |
| **Search Strategy When Low** | ‚Ä¢ Can segment market testing<br>‚Ä¢ Independent technology development<br>‚Ä¢ Player-specific optimization | ‚Ä¢ Can test anywhere in chain<br>‚Ä¢ Flexible development paths<br>‚Ä¢ Location-independent scaling |
| **Parallel Search Value** | Moderate $k_h$ optimal:<br>‚Ä¢ Too low: misses synergies<br>‚Ä¢ Too high: over-constrained | Higher $k_v$ favors parallel:<br>‚Ä¢ Breaks temporal constraints<br>‚Ä¢ Enables simultaneous chain testing |
| **Representative Example** | Tesla-Hyundai battery tech:<br>‚Ä¢ Shared charging standards<br>‚Ä¢ Common supplier networks<br>‚Ä¢ Market interdependencies | Steam engine development:<br>‚Ä¢ Material improvements<br>‚Ä¢ Manufacturing capabilities<br>‚Ä¢ Sequential refinements |
| **Macroinvention Impact** | Still constrained by complementary needs:<br>"Need metallurgy AND boring tech" | Can break through constraints:<br>"Create new species, new path" |

Key Insight: The optimal search strategy requires balancing:
1. Horizontal correlation ($k_h$): Want enough for cross-fertilization but not so much it over-constrains
2. Vertical correlation ($k_v$): Higher values increase value of parallel search to break temporal constraints

This supports Mokyr's argument that macroinventions work best with:
- Moderate horizontal correlation (enabling complementary technologies)
- High vertical correlation (breaking through historical constraints)
