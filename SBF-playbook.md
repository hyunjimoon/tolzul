# SBF Playbook

## Short Pattern (daily)
```
👁️ see → 🧠 predict → 🥷 act
```

**Drill:**
1. Input 감지
2. Expected와 비교
3. Gap에 반응

## Long Pattern (strategic)
```
학계: 관찰 → 이론 → 논문
사업: 신호 → 전략 → 실행
성찰: 감정 → 통찰 → 변화
창작: 자극 → 구조 → 작품
```

## Telltale Signs

**학계:** 논문 트렌드, 방법론 gap  
**사업:** 시장 움직임, 고객 신호  
**성찰:** 반복 패턴, 감정 변화  
**창작:** 영감 순간, 리듬 변화

## Muscle Memory

반복 → 자동화 → "see-act"
