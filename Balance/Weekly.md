## 📊 최근 주간 회고

```dataview
TABLE WITHOUT ID
	file.link as "주차",
	file.ctime as "작성일"
FROM "Balance/주별 멜로디"
WHERE file.name != "Weekly"
SORT file.ctime desc
LIMIT 8
```
