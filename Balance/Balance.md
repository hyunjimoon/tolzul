# ✋ Balance - 손 / Control

> **리듬을 조율한다. 여백의 폭 V를 조절한다.**

---

## 🥁 시간의 리듬

Balance는 매일의 박자를 기록하는 곳이다.  
시간을 분절하며, 여백 V를 안정화한다.

## 🎼 Balance 사용법

**매일**:
- `Cmd-d`로 Daily note 생성
- 아침: 오늘의 의도
- 저녁: 오늘의 Vscore

**매주**:
- 주말에 Weekly 작성
- 무엇을 배웠는가?
- 무엇을 조율할 것인가?

**매달**:
- 분기마다 Monthly 회고
- 패턴 발견
- 다음 달 방향 설정
