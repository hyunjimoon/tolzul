# ☁️ 월별 화음 - 氣 (수증기)

> *"전체 그림, 큰 흐름, 자유로운 조망"*

---

## 수증기의 속성

**자유로움**: 경계 없이 이동  
**조망**: 전체를 한눈에  
**연결**: 모든 것을 통합

---

## ☁️ 매월 한 가지

```
☁️ 이번 달 시도할 것 하나
```

**질문**:
- 이번 달 전체 그림은?
- 새로 시도할 것은?
- 큰 변화는?

---
